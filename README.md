## Development Setup

1. Clone the repository:

```bash
git clone https://github.com/bkanhu/portfolio-2025.git

cd portfolio-2025
```

2. Install dependencies:

```bash
npm install
# or
yarn install
```

3. Run the development server:

```bash
npm run dev
```
